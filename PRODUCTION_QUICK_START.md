## Step 4: Configure Environment

```powershell
# Copy example environment file
Copy-Item .env.example .env

# Edit .env and add your keys
notepad .env
```

Required settings in `.env`:
```env
INFURA_API_KEY=your_infura_key
TESTNET_PRIVATE_KEY=your_64_char_hex_key
```

---

## Step 5: Deploy to Testnet

```powershell
# Deploy to Sepolia testnet
.\omega.exe deploy sepolia --contract=target/evm/SimpleToken.sol --key=.keys/test.key

# Or with environment variable
.\omega.exe deploy sepolia --contract=target/evm/SimpleToken.sol
```

## Project Configuration

Create `omega.toml` in your project root:

```toml
[project]
name = "my-project"
version = "1.0.0"

[build]
sources = ["src", "contracts"]
targets = ["evm", "solana"]
mode = "release"

[deploy]
default_network = "sepolia"
```

- [ ] Run security audit: `.\omega.exe audit`
- [ ] Test on testnet first

- [ ] Check gas estimates
- [ ] Verify contract on explorer
- [ ] Consider external audit for high-value contracts

## Troubleshooting

### Compilation fails
```powershell
# Check syntax
.\omega.exe compile --check your_file.omega

# Verbose output
.\omega.exe compile your_file.omega --verbose
```

### Deployment fails
```powershell
# Check balance
.\omega.exe balance <your_address> --network sepolia

# Dry run first
.\omega.exe deploy sepolia --contract=... --dry-run
```

### Need help?
```powershell
.\omega.exe --help
.\omega.exe deploy --help
```

## Example Workflow

```powershell
# 1. Create new project
mkdir my-token && cd my-token

# 2. Initialize
.\omega.exe init

# 3. Write contract (or copy example)
Copy-Item ..\examples\simple_token.omega .\src\

# 4. Compile
.\omega.exe build

# 5. Test locally
.\omega.exe test

# 6. Deploy to testnet
.\omega.exe deploy sepolia --contract=target/evm/SimpleToken.sol

# 7. Verify on explorer
# Visit: https://sepolia.etherscan.io/address/<your_contract>
```
