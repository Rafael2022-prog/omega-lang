# OMEGA Language Support for VS Code

This extension provides comprehensive language support for the OMEGA blockchain programming language.

## Features

- Syntax highlighting for .mega and .omega files
- Language server integration
- Code completion and IntelliSense
- Error diagnostics and linting
- Code formatting
- Project management tools
- Cross-compilation support
- Integrated testing framework
- Documentation generation
- Security scanning
- Performance analysis

## Commands

- `OMEGA: Compile File` - Compile current OMEGA file
- `OMEGA: Build Project` - Build entire OMEGA project
- `OMEGA: Run Tests` - Execute OMEGA test suite
- `OMEGA: Deploy to Blockchain` - Deploy contracts to blockchain
- `OMEGA: Create New Project` - Initialize new OMEGA project
- `OMEGA: Add Compilation Target` - Add new compilation target
- `OMEGA: Show Available Targets` - Display compilation targets
- `OMEGA: Format Document` - Format OMEGA code
- `OMEGA: Run Diagnostics` - Run code diagnostics
- `OMEGA: Generate Documentation` - Generate project documentation
- `OMEGA: Security Scan` - Perform security analysis
- `OMEGA: Performance Analysis` - Analyze code performance

## Requirements

- VS Code 1.74.0 or higher
- Node.js 16.0 or higher
- OMEGA compiler installed globally

## Installation

1. Install the extension from VS Code Marketplace
2. Ensure OMEGA compiler is available in PATH
3. Open any .mega or .omega file to activate the extension

## Configuration

The extension can be configured through VS Code settings:

- `omega.compiler.path`: Path to OMEGA compiler
- `omega.linter.enabled`: Enable/disable linting
- `omega.formatter.enabled`: Enable/disable formatting
- `omega.server.trace`: Enable language server tracing
